Projects

Bachelor thesis: Manipulator with parallelogram linkages

Feb 2021 – Jun 2021

In this project – “Analysis and development of a parallelogram linkage manipulator”, I worked on building a control system for the robot which served as a palletizer. In such repetitive operations, performance can be improved by adjusting the control signal each time the robot repeats its motion. This method is called Iterative learning control (ILC) and does not require that the disturbances be known or measured. While this multi-articulated robot is difficult to be accurately modelled, the model using ILC is simple enough to be implemented conveniently in real-time control using a STM32 MCU.

Research: 5-dof manipulator

Sep 2020 – Jun 2021

In this project – “Research on modelling and trajectory planning for a 5-DOF robot manipulator”, we developed a MATLAB program to simulate and control the robotic arm. Forward kinematics is derived using Denavit-Hartenberg method, and inverse kinematics is calculated using algebraic and geometric methods. We formulated a trajectory planner that can generate various motion profiles (polynomial, trapezoidal and S-curve) and automatically select the most time-optimal path. Our project makes a convenient system for algorithm evaluation by the use of MATLAB for both simulation and real-time experiment purpose.

Research: Line-following robot

Sep 2020 – Jan 2021

In this project – “Design and implementation of a line-following robot”, we designed, built, simulated and tested a line-following robot given a specific reference path comprising a combination of straight/curved segments and intersections at which the robot should turn. We aim to create a robot that is accurate, robust and swift while optimizing the design to be cost-effective. The control law is proposed and tuned such that it stabilizes the error dynamics of the system, and is implemented in a distributed network of PIC MCUs.

Research: Yehchai – sorting waste with AI

Jan 2020 – Jun 2020

“YEHCHAI Project: An AI Waste Sorter” presents our work on researching, designing and implementing a machine that utilizes artificial intelligence in sorting waste at the source. To be able to replace conventional labelled bins, YEHCHAI should be effective in classifying and storing, attractive to end users, robust, while keeping cost and maintenance at the minimum. The first prototype was tested in VNU-HCMUT campus and out of 30 surveyed users, 73.3% absolutely preferred our product to conventional labelled bins.
